System Crasher

This C program, designed by Mr. Linux, demonstrates my proficiency in programming and system automation. The program offers a simple menu-driven interface that performs a variety of tasks. When executed, the user is presented with three main options: crashing the system, accessing contact information, or exiting the program. Selecting the system crasher initiates a function designed to repetitively open a web browser, effectively consuming system resources. The contact information option provides links to my GitHub, Instagram, and LinkedIn profiles, allowing users to easily connect with me on various platforms. This program showcases my skills in creating functional and interactive command-line tools, emphasizing my dual expertise in both ethical hacking and programming.

Port Scanner

Welcome to the Port Scanner tool powered by Python's Tkinter! This application allows you to scan a range of ports on a specified host or IP address. Enter the target host, starting port, and ending port numbers, then click "Scan Ports" to initiate the scan. The results will display whether each port in the specified range is open or closed. You can clear the results at any time using the "Clear Results" button. This tool is useful for network administrators and security professionals to identify open ports for troubleshooting or security auditing purposes.
